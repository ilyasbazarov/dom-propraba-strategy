## 4. Связь с проектом стратегии

- База для **runway floor**: OPEX-компонента есть (черновик: ~1.38М/мес по марту–маю).
  Не хватает остатка казны (DQ-7) и решения владельца по значению N.
- Калибратор аппетитов ставок L2: свободный операционный денежный поток пока
  отрицательный/около нуля — ставки проверяются против вливаний и роста дохода,
  не против накопленной подушки.
- Цифра «выручка ~6М/мес» из артефактов прошлого чата устарела:
  фактический диапазон 3.3–7.6М/мес с восходящим трендом.
