## Задача

Разобрать артефакты 6–9 на атомарные тезисы и разложить каждый тезис
в одну из корзин:

| Корзина | Что кладём | Формат выхода |
|---|---|---|
| **B1 → S06-кандидаты** | Принятые в прошлом чате решения (например R-01…R-07, запреты, «две раздельные программы лояльности») | Таблица: тезис · источник (файл+секция) · предлагаемая метка IRREVERSIBLE/REVERSIBLE · зависит ли от устаревших цифр |
| **B2 → ASSUMPTION-реестр S07** | ВСЕ цифры из артефактов (GMROI, SKU-каунты, сегменты, бюджеты, LTV, ROI-оценки) | Таблица: цифра · источник · оценка риска при ложности (HIGH/MED/LOW) · чем проверяется |
| **B3 → S02-кандидаты** | Векторное/ставочное сырьё (программы, каналы, механики) | Таблица: тезис · источник · предлагаемый слой (L1/L2/L3) с одним предложением обоснования |
| **B4 → S10-кандидаты** | Вопросы и неясности, НЕ покрытые существующими Q-01…Q-18 | Таблица: вопрос · источник · что блокирует |
| **B5 → S05/S09-кандидаты** | Правила работы и термины, отсутствующие в S05/S09 | Список с источниками |
| **B6 → GRAVEYARD-кандидаты** | Тезисы, очевидно нерелевантные стратегии (чисто операционный псевдокод аппендикса B и т.п.) | Одна строка на тезис с причиной |

## Что НЕ делать

- НЕ принимать решений: все корзины — кандидаты, финальную раскладку делает
  архитектор/владелец.
- НЕ оценивать качество идей (это T-S2), НЕ улучшать формулировки — инвентаризация.
- НЕ переносить цифры в S01 (запрещено правилом S01 §Правила п.3).
- НЕ редактировать файлы репо: выход сессии — один md-отчёт.
- НЕ дублировать существующие Q-01…Q-18 из S10.
- При любой неоднозначности (тезис не лезет ни в одну корзину, конфликт
  источников) — CONTEXT GAP, не догадка.

## Формат выхода

Один файл `briefs/OUT-001_migration_report.md`:
1. Секции B1–B6 (таблицы выше).
2. **Контрольные цифры:** тезисов извлечено всего / по каждому артефакту /
   по каждой корзине. Сумма по корзинам = сумме извлечённого.
3. Session-блок по S05 §7.

## Приёмка (проверяет архитектор)

1. Контрольные цифры сходятся (ничего не потеряно и не задвоено).
2. Каждая цифра из артефактов присутствует в B2 — выборочная проверка
   по 10 случайным цифрам из источников.
3. Ни один тезис не «улучшен» — выборочная сверка формулировок.
4. B4 не дублирует S10.
5. Session-блок присутствует и полон.
